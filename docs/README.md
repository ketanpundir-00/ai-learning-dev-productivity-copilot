# Project Documentation

This folder contains all core documentation and presentation materials for the  
**AI Learning & Developer Productivity Copilot** project.

## Contents

- **requirements.md**  
  Defines the problem statement, user stories, functional and non-functional requirements.  
  Generated using **Kiro** as part of the AI for Bharat Hackathon workflow.

- **design.md**  
  Describes the system architecture, AI components, data flow, scalability, and responsible AI considerations.  
  Generated using **Kiro**.

- **pitch-deck.pdf**  
  The final presentation deck explaining the problem, solution, AI impact, and relevance to Bharat.

## Notes

- These documents are created for **design clarity and project explanation**
- This folder is intended for **reviewers, mentors, and recruiters**
- The repository is maintained for **portfolio and showcase purposes**
